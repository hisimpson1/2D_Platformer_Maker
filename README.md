# 2D_Platformer_Maker
유니티로 2D 플랫포머 게임을 만들수 있는 툴을 개발합니다.

이 프로젝트는 Unity Asset Store 에셋을 사용합니다.
해당 에셋은 라이선스 문제로 저장소에 포함되어 있지 않습니다.

- 구개인 홈페이지:     https://dolphin.ivyro.net/file/alllist.php?to=mobile
- 신규 개인 홈페이지:   https://hisimpson1.github.io/
