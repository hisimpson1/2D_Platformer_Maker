# 2D_Platformer_Maker
유니티로 2D 플랫포머 게임을 만들수 있는 에디팅 툴입니다.

이 프로젝트는 Unity Asset Store 에셋을 사용합니다.
해당 에셋은 라이선스 문제로 저장소에 포함되어 있지 않습니다.
에디터에서 실행할려면 다음에셋이 필요합니다.

- 심플 2D 플랫포머 에셋 팩 BE2 (퍼블리셔: 골드메탈 스튜디오) : https://assetstore.unity.com/packages/2d/characters/simple-2d-platformer-assets-pack-188518
- FREE Casual Game SFX Pack : https://assetstore.unity.com/packages/audio/sound-fx/free-casual-game-sfx-pack-54116

[다운로드](https://github.com/hisimpson1/gittest/raw/master/unity/2D_Platformer_Maker/2D_Platformer_Maker.zip)

# 기능 : unity platformer 런타임 편집기능 추가
- 유니티 버전 : 6000.0.64f1
- Player Idle, Walk 애니메이션 연결, 좌, 우로 이동, 점프 구현
- 런타임 Load, Save, 타일 액터 에디팅 기능 구현
- 편집은 컴맨드팬턴으로 Undo, Redo 기능 구현
